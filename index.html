Basic LESS demo

Basic less is not very hard

  1. Install LESS using npm "npm install -g less"
  2. Create a project
  3. Choose your IDE
  4. Install grunt or gulp globally for tasks "npm install -g gulp"
  5. Install gulp and gulp-less locally "npm install gulp gulp-less"
  6. Create gulpfile and tasks
  7. ...
  8. Profit!
